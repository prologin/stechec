\section{Le projet TowBowlTactics}

\frame
{
  \begin{block}{Présentation de Blood Bowl}
    \begin{itemize}
      \item Jeu de plateau édité par Games Worshop depuis 1986.
      \item Figurines, différentes races.
      \item Allie convivialité, rapidité, hasard et stratégie.
    \end{itemize}
  \end{block}
}

\frame
{
  \begin{block}{Blood Bowl tactics}
    \begin{itemize}
      \item Adaptation vidéo de Blood Bowl.
      \item Née de la volontée d'une personne, Pascal Bourut, en 2001.
      \item Programmé en C++/SDL, réalisé en 1 an.
      \item Communautée se formant autour du jeu, mais pas
        suffisamment de programmeur pour le faire vivre.
    \end{itemize}
  \end{block}
}

\frame
{
  \begin{block}{Légalement}
    \begin{itemize}
      \item Projet entièrement en GPL.
      \item Droits d'adaptation vis-à-vis de GW flou.
    \end{itemize}
  \end{block}
}

\frame
{
  \begin{block}{Reprise fin 2005}
    \begin{itemize}
      \item Reprise amorcée par un fan: Mathieu.
      \item Soutient technique de Nekeme prod.
      \item Mise en place rapide d'une solide infrastructure:
        \begin{itemize}
          \item site web,
          \item forum,
          \item wiki,
          \item subversion,
          \item mailing-listes.
        \end{itemize}
    \end{itemize}
  \end{block}
}

\frame
{
  \begin{block}{Organisation de l'équipe}

      \item Pas d'organisation, à l'arrach.
      \item C'est mieux comme ca.
      \item Document de base décrivant du point de vue technique le
        projet.
    \end{itemize}
  \end{block}
}

\frame
{
  \begin{block}{Choix technologique}
